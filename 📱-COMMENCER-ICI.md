## ⚡ TL;DR (Trop Long; Pas Lu)

```bash
# 1. Configurer votre IP
ipconfig getifaddr en0
# Éditer capacitor.config.ts ligne 13

# 2. Sync
npm run mobile:sync

# 3. Lancer serveur (Terminal 1)
npm run dev

# 4. Lancer app (Terminal 2)
npm run mobile:run
```

**C'est tout ! 🎉**
